HTML Text Formatting

Basic Paragraphs

This is a standard paragraph of text. It's the most common way to display text on a webpage. You create it using the <p> tag.

Headings

HTML provides six levels of headings, from <h1> to <h6>:

This is an h2 heading

This is an h3 heading

This is an h4 heading

Emphasis and Importance

You can make text bold and important using the <strong> tag.

You can emphasize text using the <em> tag (usually renders as italics).

Sometimes you just want to make things bold without adding importance using the <b> tag, or italic without adding emphasis using the <i> tag.

Other Text Elements

You can highlight text using the <mark> tag.

You can show deleted text and inserted text.
